
## ğŸ†˜ NEED HELP?

### Stuck on Something?

1. **Can't get local running:**
   - Check Node.js installed: `node --version`
   - Check dependencies: `npm install`
   - Check Firebase variables

2. **Render deployment fails:**
   - Check environment variables
   - Verify FIREBASE_PRIVATE_KEY has `\n`
   - Check GitHub push succeeded
   - Review Render logs

3. **Extension won't package:**
   - Ensure all files present
   - Check manifest.json valid
   - Verify icons are PNG format
   - Run build script with admin

4. **Chrome Web Store rejection:**
   - Read rejection email carefully
   - Fix the issue
   - Update version number
   - Resubmit
